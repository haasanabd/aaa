# Haa Management - النسخة الاحترافية 2026 (الخزنة الآمنة)

تطبيق متكامل لإدارة ونسخ الصور والفيديوهات بخصوصية مطلقة، مصمم ليعمل كخزنة مخفية تماماً عن نظام الهاتف.

## الميزات الجديدة في هذا التحديث:

1.  **نظام الخزنة المخفية (Safe Vault):**
    *   يتم تخزين جميع الملفات في مجلد خاص بالتطبيق (`Internal Storage`).
    *   **الخصوصية:** المجلد مخفي تماماً عن معرض الصور (الاستوديو) وعن أي تطبيق آخر في الهاتف.
    *   تم إضافة ملف `.nomedia` لضمان عدم قيام النظام بمسح أو عرض هذه الملفات.

2.  **قاعدة بيانات رشيقة (Ultra-Light DB):**
    *   قاعدة البيانات تخزين فقط "المعلومات الوصفية" وصور مصغرة فائقة الضغط.
    *   هذا يجعل حجم قاعدة البيانات صغيراً جداً (حوالي 100 ميجابايت لـ 10,000 صورة) وسريعاً جداً في التصفح.

3.  **التحديد المتعدد والنسخ الجماعي:**
    *   إمكانية اختيار عدد غير محدود من الصور والفيديوهات دفعة واحدة.
    *   نظام معالجة خلفي يضمن عدم تعليق التطبيق أثناء الحفظ.

4.  **نظام النسخ الاحتياطي الشامل:**
    *   زر "نسخ احتياطي" يقوم بضغط الخزنة الآمنة بالكامل مع قاعدة البيانات في ملف واحد.
    *   زر "استعادة" يقوم بإعادة بناء الخزنة بالكامل كما كانت.

5.  **تصدير الملفات للهاتف:**
    *   يمكنك في أي وقت تحديد ملفات من داخل الخزنة وإعادتها للمعرض أو مشاركتها عبر تطبيقات أخرى.

## طريقة التشغيل:
1.  تأكد من تثبيت Flutter SDK.
2.  نفذ الأمر `flutter pub get`.
3.  قم ببناء التطبيق باستخدام `flutter build apk`.

## ملاحظة هامة:
عند استعادة نسخة احتياطية، يفضل إغلاق التطبيق وإعادة فتحه لضمان تحديث الاتصال بقاعدة البيانات المستعادة.
